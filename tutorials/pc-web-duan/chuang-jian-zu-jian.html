创建组件

创建组件

VD 可创建三种类型的组件：

  • 通用组件
  • 业务组件
  • 模块

通用组件

不带业务数据的、可复用的组件，比如，时间选择控件。

通用组件具有以下属性：

字段 说明
名称 组件的名称，首字母需大写
描述 组件的描述
版本号 组件的版本号
兼容性 组件的兼容性
资源目录 组件可包含的资源目录，可选值有 图片、样式、静态数据、单元测试用例、使用文档；
组件类型 React 组件的类型，可选值有 Class 写法、Function 写法；默认值为：Class 写法
组件的生命周期 React 组件生命周期；Function 类型的组件无此项。
属性类型验证 React 组件属性类型验证，开启属性验证就会有默认属性容器
其他说明 其他信息

业务组件

可以包含通用组件，带业务流程、数据的组件。

业务组件除了拥有通用组件的属性外，还具有以下属性：

字段 说明
请求地址 HTTP 请求的 URL，只能使用相对路径格式，比如：/api/v1/getData
请求地址的变量名 通过 JS 变量调用请求地址的变量名称。比如：getData
请求方式 HTTP 请求的方式，可选值有：GET、POST、DELECT、PUT、PATCH；默认值为：GET

模块

库类型项目最新单元，比如，方法。

模块属性同通用组件区别如下：

字段 同通用组件的区别
名称 模块名称首字母小写
资源目录 没有图片选项
组件的类型 无
组件的生命周期 无
属性类型验证 无
