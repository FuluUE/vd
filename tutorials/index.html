自述

前端可视化开发平台

前端项目（React，etc...）源码管理平台

前端可视化开发平台（VD，Visual Development）致力于通过更简单的方式来管理你的项目源码（前端类项目），包括 编译、预处理、压缩、混淆、合并、模块化、按需加载等等处理。

她也期望成为：

  • 前端“新技术”落地平台
  • 前端编码规范化平台
  • 前端可持续集成环境
  • 前端项目脚手架工具
  • 前端创新产品研发平台

她期望包含丰富的前端业务形态项目脚手架（PC Web 端、移动 Web 端、库、PC 客户端、移动客户端、...）和更广的组件类型：通用组件（eg：控件）、业务组件（eg：业务流程）、模块（eg：方法）。

她还致力于帮你写出更出色的代码，比如项目样板、代码片段、疑难点解决方案等功能。

平台特色

特色 说明
React 技术栈集成 集成 Webpack(v3.x), Babel(v6.x), React(v16.x), React Router(v4.x), Dva(v2.x), Less(v2.x), PostCSS(v6.x), Axios(v0.x), Mockjs(v1.x), Jest(v21.x), Enzyme(v3.x), ESLint(v4.x), Ant Design(v3.x), Ant Design Mobile(v2.x)...
自动更新 总能使用最新的，全部的功能，且是免费的。
双重开发模式 可视化 和 命令行 双重开发模式，总有一款适合你。
规范化开发 约定优于配置，让你享受无感知的编码规范化。
组件化管理 高可复用性的代码管理方式，你值得拥有。
项目样板 自定义配置，规范而不失灵活。
实时调试 还在不停的切换屏幕，F5 刷新？试试热加载吧。
一键部署 远程登录？*TP？ 不，一个按钮轻松搞定。
代码片段 还在为代码写的不够好发愁？来看看最佳实践吧。
远程代理 拦截请求，随机响应，脱离后端开发。

等等，这一切，以及更多的功能，期待你去发现。
